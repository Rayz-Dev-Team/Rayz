GET ENDPOINTS 
Admin dashboard:

GET /stats Returns a number of servers Rayz is in, and a number of users Rayz can see.

GET /server/SERVER_ID/info This returns all the data needed for a website to display. Such as: staff member information, bots, server information.

GET /token/USER_ID/generate Generates a Token and Password for authentication.

GET [PROTECTED] /server/SERVER_ID/settings Generates a Token and Password for authentication.

GET [PROTECTED] /server/SERVER_ID/channel/CHANNEL_ID Checks if a channel is valid. Also returns channel ID/Name.

GET [PROTECTED] /server/SERVER_ID/bot-permissions Returns a dict list of booleans for specific permissions.
POST ENDPOINTS 
Admin dashboard:

POST /token/USER_ID/validate Gets a response if a Token is valid or not. Will return code errors for: Invalid User ID, Token, etc.
