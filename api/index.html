Rayz API Documentation!

/stats · [GET] · Unprotected

Returns a number of servers Rayz is in, and a number of users Rayz can see.

curl --request GET https://api.rayzbot.xyz/stats

/server/SERVER_ID/info · [GET] · Unprotected

This returns all the data needed for a website to display. Such as: settings, staff member information, bots, server information.

curl --request GET https://api.rayzbot.xyz/server/SERVER_ID/info

/token/USER_ID/generate · [GET] · Unprotected

Generates a Token and Password for authentication.

curl --request GET https://api.rayzbot.xyz/token/USER_ID/generate

/token/USER_ID/validate · [POST] · Unprotected

Gets a response if a Token is valid or not. Will return code errors for: Invalid User ID, Token, etc.

curl -X POST -H "Content-Type: application/json" -d "{\"token\": \"TOKEN HERE\"}" https://api.rayzbot.xyz/token/USER_ID/validate

/server/SERVER_ID/settings · [GET] · Protected

Gets Rayz's settings for the server.
