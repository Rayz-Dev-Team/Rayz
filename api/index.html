API Documentation

/stats · [GET] · Unprotected

Returns a number of servers Rayz is in, and a number of users Rayz can see.

curl --request GET https://api.rayzbot.xyz/stats

/server/SERVER_ID/info · [GET] · Unprotected

This returns all the data needed for a website to display. Such as: settings, staff member information, bots, server information.

/token/USER_ID/generate · [GET] · Unprotected

Generates a Token and Password for authentication.
